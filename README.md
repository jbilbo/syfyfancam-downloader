# Syfyfancam Downloader
[![Build Status](https://travis-ci.org/jbilbo/syfyfancam-downloader.svg?branch=master)](https://travis-ci.org/jbilbo/syfyfancam-downloader)

Uses syfyfancam gem to download the syfyfancam.com photo files into a directory in your computer.

usage:

```shell
ruby bin/syfyfancam_downloader http://www.syfyfancam.com/videos/ojt1nd5bnbog/
```
